Cách Sử dụng hàm visualize.py

```bash
git clone https://github.com/longle0125/hnswlib.git # hoặc pull nếu đã có sẵn

# Tạo môi trường ảo
python -m venv venv
# Kích hoạt môi trường
venv\Scripts\activate
pip install -r requirements.txt
#sử dụng thư viện hnswlib đã được modify
cd hnswlib
pip install .
```
